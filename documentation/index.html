Getting Started

Welcome to TechFit React Next.js template! We would like to thank you, for buy our template - TechFit.

TechFit is a React Next.js and Tailwind CSS based Coaching & Online Courses template. This has been built with React and as framework Next.js!

Setup Development Environment

To kickstart the development of the web app with TechFit, at first need to setup the react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

Preparation

You'll need to install Node.js (Recommended Version) (NPM comes along with it) and serve (a NPM package)

Use command line tool i.e. PowerShell or terminal and navigate to the project(Plaon) root. This is PowerShell in windows and terminal on mac/linux.

Let's have a look on the setup process:

Use terminal and navigate to the project root.

Then run : npm install

Then run : npm run dev

It'll return something like this in the console:

> project@1.0.0 dev drive:\project

> next

[ wait ] starting the development server ...

[ ready ] started server on http://localhost:3000 ...

[ event ] compiled successfully

Now, in the browser go to localhost:3000

For Production Build

Run : npm run build

Then Run : npm run start

See the official next.js doc here
